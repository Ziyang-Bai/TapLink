TapLink是一款基于Python的文件传输工具，它使用TCP协议进行文件传输，计划支持断点续传和文件校验。TapLink可以用于局域网内文件传输，也可以用于远程文件传输。
## 功能
- 文件传输 [X]
- 断点续传 [O]
- 文件校验 [O]
